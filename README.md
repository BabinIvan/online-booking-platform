# online-booking-platform
## Описание проекта
Система онлайн-бронирования для гостиниц, ресторанов, конференц-залов и мероприятий.

## Используемые технологии и инструменты
GitHub
Git
Jira
Visual Studio Code
README.md

## Системные требования
- Операционная система Windows 10/11
- Веб-браузер (Google Chrome, Microsoft Edge)
- Подключение к Интернету
- Visual Studio Code

## Основные функции
- Онлайн-бронирование
- Календарь доступности
- Онлайн-оплата
- Email и SMS уведомления
- Панель администратора
- Управление поставщиками услуг

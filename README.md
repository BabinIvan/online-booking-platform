# online-booking-platform
## Описание проекта
Система онлайн-бронирования для гостиниц, ресторанов, конференц-залов и мероприятий.

## Используемые технологии
- Next.js
- NestJS
- PostgreSQL
- Stripe API
- React Native

## Системные требования
- Node.js 22+
- PostgreSQL 16+
- Git
- Visual Studio Code

## Основные функции
- Онлайн-бронирование
- Календарь доступности
- Онлайн-оплата
- Email и SMS уведомления
- Панель администратора
- Управление поставщиками услуг

## Установка

```bash
git clone https://github.com/BabinIvan/online-booking-platform.git
cd online-booking-platform
npm install
npm run dev

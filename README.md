# online-booking-platform
## Описание проекта

Создание системы бронирования для нескольких поставщиков услуг (гостиницы, рестораны, конференц-залы, мероприятия) с отдельным порталом для каждого бизнес-клиента и мобильным приложением для конечных пользователей.

## Используемые технологии

* Next.js
* NestJS
* PostgreSQL (multi-tenant)
* Stripe API
* React Native
* GitHub
* Git
* Jira
* Visual Studio Code

## Ключевые функции

* Мультиарендная архитектура с изоляцией данных для каждого клиента
* Интерактивный календарь с доступностью в режиме реального времени
* Онлайн-оплата с автоматическим формированием счетов
* Уведомления о подтверждении, отмене и напоминании через SMS и e-mail
* Панель администрирования для каждого поставщика услуг

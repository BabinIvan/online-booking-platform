# online-booking-platform
## Описание проекта
Система онлайн-бронирования для гостиниц, ресторанов, конференц-залов и мероприятий.

## Используемые технологии и инструменты

- GitHub — хранение репозитория и контроль версий
- Jira — Product Backlog, Scrum-доска и спринты
- Visual Studio Code — среда разработки
- README.md — описание проекта

## Системные требования
- Node.js 22+
- PostgreSQL 16+
- Git
- Visual Studio Code

## Основные функции
- Онлайн-бронирование
- Календарь доступности
- Онлайн-оплата
- Email и SMS уведомления
- Панель администратора
- Управление поставщиками услуг
